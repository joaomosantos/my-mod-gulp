# My Modulo Gulp

### Usage / Install

```js
npm install
```

### Basic Commands

```js
// Compile LESS to CSS
gulp compiler-less
// Compile SASS to CSS
gulp compiler-sass
// Joining in a single css (all.css)
gulp concat-css
// Minify all.css (all.min.css)
gulp minify-css
// Start BrowserSync (support SSI)
gulp serve
// To assign auto prefixer in CSS
gulp autoprefixer-css
```
